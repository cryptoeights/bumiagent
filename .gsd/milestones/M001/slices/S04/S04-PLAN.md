# S04: Frontend — Landing, Deploy Form, Chat

**Goal:** User can visit landing page, connect wallet, fill 3-field deploy form, and chat with an agent in browser.
**Demo:** Next.js app at localhost:3000 with landing page CTA, wallet connect (MetaMask/WalletConnect), agent deploy form, and working chat interface.

## Must-Haves

- Next.js 14+ App Router with TypeScript
- Wallet connect via wagmi + viem (MetaMask/WalletConnect)
- Landing page with "Launch Agent" CTA
- 3-field deploy form (name, template dropdown, price)
- Chat interface for talking to agents
- shadcn/ui components
- Distinctive visual design (not generic AI aesthetics)
