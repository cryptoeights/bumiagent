07 — DEEP DIVE

Tech Architecture

⏱ 60 seconds
Screen Pitch deck Architecture slide or README diagram
"Architecturally, Bumi Agent consists of 3 layers:"
Frontend — Next.js 16 with TypeScript and Tailwind CSS v4. Wallet connection via RainbowKit + wagmi. Analytics charts powered by Recharts. Fully mobile-responsive. Deployed on Vercel.
Backend — Hono, a lightweight Node.js framework. 16 REST API endpoints. PostgreSQL database on Supabase via Drizzle ORM. Caching and rate limiting on Upstash Redis. LLM gateway through OpenRouter — one endpoint, 8 models. Payment processing via x402 protocol. Wallet encryption using AES-256-GCM. Deployed on Railway.
Blockchain — 3 smart contracts on Celo Mainnet, written in Solidity 0.8.25 with Foundry. Using OpenZeppelin libraries — ERC-721, ReentrancyGuard, Ownable. 85 unit tests, full coverage. All verified on Celoscan.
